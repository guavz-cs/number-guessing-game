# Number Guessing Game

This is a game where a user will want to guess a secret number so that they can try to find the correct answer using simple hints.

## Game Flow

The computer will pick a random number between 1 and 100 then the player will type a guess. The application then compares the user input with the computer's number and either says "Too High", "Too Low" or "Correct". The game ends when the users guesses correctly.
